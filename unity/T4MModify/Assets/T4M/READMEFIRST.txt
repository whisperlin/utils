If you want use the SC version on a already in Black Edition Scene, you must to use the file you can find in the "_T4M BE to SC" folder of T4M folder

After installation a new menu will appear : T4MTool
- Clic and launch the process.
- After that, Deactivate and reactive the LOD and Billborad in the T4M menu.
- Delete the .dll files of T4M Black Edition
- Delete the "T4MBEtoSC" file in T4M/Editor folder

Your scene is ready to work with "T4M Source Codes"
